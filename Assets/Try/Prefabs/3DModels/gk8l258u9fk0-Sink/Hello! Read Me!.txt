- Bathroom Sink -

Models are Game-Ready.

It is a polygonal model (quads and tris), scaled and represented as the original real life object. Built in real world size (units - cm).

Model has 10 objects within, which are grouped to one group. For easy use objects are named logically (using English language).

Models are set in world coordinates x0 y0 z0.

No special plugin is needed to open scene.

File Formats:

.MAX 2016 
.FBX
.OBJ
.3DS

Due to the nature of these formats, there may be some textures that have to be loaded by hand and possibly triangulated geometry.
